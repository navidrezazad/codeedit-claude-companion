# CodeEdit Remote iOS

This is the SwiftUI iOS companion app for the CodeEditV2 terminal bridge.

The iOS app discovers the Mac bridge over Bonjour using `_codeeditv2-term._tcp`, authenticates with the passcode configured in the Mac app, lists active terminal sessions, attaches to a selected session, mirrors terminal output, and sends command input back to the Mac.

After a successful pairing, the app can save both local-network connection details and the Mac bridge's detected public IP details. Public IP reconnect still requires a reachable path such as VPN, Tailscale, router forwarding, or another secure tunnel.

## Tabs

- Terminal: shows a compact text mirror of the selected Mac terminal session and sends command input to that session.
- MD Stream: renders the Markdown stream file generated by the Mac bridge and exposes Markdown update/rewrite controls.

## Markdown Rendering

The Markdown view uses bundled local assets instead of CDN dependencies:

- `marked.min.js` for Markdown parsing
- `purify.min.js` for HTML sanitization
- `katex.min.js` and `katex.min.css` for math between `$...$`, `$$...$$`, `\(...\)`, and `\[...\]`

The view prewarms a hidden renderer after authentication to reduce first-open lag.

## Claude Helpers

The Mac bridge can call Claude for two Markdown workflows:

- Update: send the newest terminal increment and append a Markdown-formatted stream entry.
- Rewrite: send the full current Markdown document plus a user prompt and replace the stream file with the returned Markdown.

The Claude process is launched by the Mac bridge as `claude -p --model claude-opus-4-8 --effort medium` with tool use disabled. The bridge writes the returned Markdown content after the process exits.
